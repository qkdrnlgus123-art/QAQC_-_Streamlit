# ⚙️ FDC Dashboard — 반도체 금속 식각 공정 이상탐지

반도체 금속 식각(Metal Etching) 공정에서 발생하는 웨이퍼 이상을 실시간으로 탐지하고 원인을 분석하는 대시보드입니다.

---

## 📋 주요 기능

### Tab 1 · 공정 현황
- 전체 웨이퍼 수, 이상 웨이퍼 수, 이상 발생률 KPI 표시
- SPE 시점별 시계열 차트로 웨이퍼 단일 분석
- Fault 그룹별 분포 (TCP / RF / 압력 / 가스 / He Chuck)

### Tab 2 · 이상 웨이퍼 목록
- 이상 탐지된 웨이퍼 목록 (SPE 기준 내림차순)
- Fault 유형·등급별 필터링 (위험 / 주의 / 확인)
- 선택 웨이퍼의 조치 권고 (긴급 / 우선 / 확인 단계)

### Tab 3 · 웨이퍼 처리 이력
- 전체 웨이퍼를 처리 순서대로 카드 형태로 시각화
- 연속 이상 감지 (앞뒤 2개 윈도우 내 이상 2개 이상 → 챔버 불안정 경보)
- SPE 시점별 이탈 그래프 — 점 클릭 시 해당 시점 신호 기여도 표시
- 센서 그룹별 파이차트 (RFM / OES / EV) + 개별 변수 Top 10 bar chart

---

## 🧠 모델 설명

### MPCA 기반 차수별 MSPC

웨이퍼 데이터를 실험 차수(29 / 31 / 33)별로 독립적인 PCA 모델로 학습합니다.

```
새 웨이퍼 입력
    ↓
각 차수 PCA 모델에 투영 → SPE 최소 차수 선택 (가장 유사한 공정 조건)
    ↓
선택된 차수의 T²·SPE 한계치로 정상/이상 판단
    ↓
이상 판정 시 → 2단계 분류 (KNN 기반 Fault 유형 분류)
```

**T² (Hotelling's T²)**: 주성분 공간 내 중심으로부터의 거리 기반 이상 탐지  
**SPE (Squared Prediction Error)**: PCA 재구성 오차 기반 이상 탐지

### SPE 시점별 기여도 분석

```
웨이퍼 잔차 벡터 (21,800차원 = 218변수 × 100시점)
    ↓
res[v * N + t]²  →  v번째 변수의 t시점 기여도
    ↓
RFM / OES / EV 그룹별 합산 → 파이차트
Top 10 개별 변수 → bar chart
```
